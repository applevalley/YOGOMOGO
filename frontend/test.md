  - 그룹 추천이 현재 오류가 있음, 랜덤셔플도 안됨

  - 전반적인 css 보수 필요
  - 카카오맵으로 검색 및 길찾기 버튼 눌렀을 때, 새 창에서 검색이 가능하도록 수정 필요
